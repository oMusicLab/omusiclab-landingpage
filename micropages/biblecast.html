BibleCast

Present Scripture from the YouVersion API — search books and verses, build slides, and present offline.

Not published yet

BibleCast is a LitePresenter experiment focused on scripture slides. We are prototyping search, slide layout, and offline caching before opening it up publicly.

What we are building

YouVersion integration

Pull verse text through the YouVersion API so passages stay accurate and easy to quote on screen.

Book & verse search

Jump to a passage quickly — browse by book, enter a reference, and land on the verses you need.

Slide builder

Turn selected verses into readable slides with spacing and emphasis suited for projection.

Offline presentation

Cache prepared passages so you can present even when the network drops mid-service or mid-study.

Intended flow

  1. 1
    Find a passage Search by book, chapter, and verse through the YouVersion API.
  2. 2
    Build your slides Select verses and arrange them into a slide deck for teaching or worship.
  3. 3
    Present offline Run the deck from LitePresenter with cached text when connectivity is unreliable.
